## â“ FAQ

**Q: Will my tasks sync across devices?**
A: No, each device stores its own data locally. This keeps your data private and works offline.

**Q: What happens if I clear browser data?**
A: Your tasks and routines will be deleted. Don't clear browser data if you want to keep them.

**Q: Can I export my data?**
A: Currently data lives in browser localStorage. You can view it in DevTools but export isn't built-in yet.

**Q: Do notifications work when my phone is off?**
A: Notifications work when phone is on but locked/sleeping. They won't work if phone is powered off completely.

**Q: Can I use this on multiple browsers?**
A: Yes! Each browser maintains its own separate todo list.
