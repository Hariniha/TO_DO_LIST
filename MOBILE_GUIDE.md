### Troubleshooting Mobile Issues

**Notifications Not Working?**
- Check browser notification permissions in phone settings
- Ensure the browser tab is still open
- Make sure battery saver isn't blocking notifications
- Try refreshing the page

**Layout Issues?**
- Try landscape mode if needed
- Clear browser cache
- Update your browser to the latest version

**Performance Issues?**
- Close other browser tabs
- Clear completed tasks regularly
- Check available storage space
